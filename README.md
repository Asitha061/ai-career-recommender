# AI Career Recommendation System

## Overview

The AI Career Recommendation System is a simple web application that suggests suitable career paths based on a user's interests.

## Features

* User-friendly interface
* Career recommendations based on selected interests
* Built using HTML, CSS, and JavaScript
* Responsive and easy to use

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. Select an area of interest.
2. Click the **Recommend Career** button.
3. The system displays a suitable career recommendation.
